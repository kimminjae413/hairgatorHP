HAIRGATOR 계정 삭제

계정 삭제를 원하시면 아래 방법 중 하나를 선택해주세요.

⚠️ 계정 삭제 전 확인사항

  • 계정 삭제 시 모든 데이터가 영구적으로 삭제됩니다.
  • 삭제된 데이터는 복구할 수 없습니다.
  • 미사용 크레딧/토큰은 환불되지 않습니다.

계정 삭제 방법

방법 1: 앱 내에서 삭제

  1. HAIRGATOR 앱 실행
  2. 마이페이지 → 설정 이동
  3. "계정 삭제" 버튼 클릭
  4. 확인 후 삭제 완료

방법 2: 이메일로 요청

아래 이메일로 계정 삭제를 요청해주세요:

  • 제목: [계정삭제요청] 본인 이메일 주소
  • 내용: 가입한 이메일 주소와 삭제 요청 사유

계정 삭제 문의

삭제되는 데이터

  • 서비스 이용 기록
  • 저장된 설정 및 환경설정
  • 결제 및 크레딧 정보
